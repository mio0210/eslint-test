# ESLint Test Project

A sample JavaScript project with ESLint configured for practicing lint error detection and fixing.

## Setup

```bash
npm install
```

## Lint

```bash
npm run lint
```

## ESLint Rules

| Rule | Description |
|------|-------------|
| `no-unused-vars` | Disallow unused variables |
| `no-console` | Disallow `console` usage |
| `eqeqeq` | Require `===` and `!==` |
| `semi` | Require semicolons |
| `quotes` | Require single quotes |
| `no-var` | Disallow `var` declarations |
| `prefer-const` | Require `const` for never-reassigned variables |
